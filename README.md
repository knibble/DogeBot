> Multi-Purpose Discord Bot (Currently in Development)

## Development

### Getting Started

#### Add an .env file with this setup:

```bash
COMMAND_PREFIX=!
OWNER=
BOT_ID=
BOT_TOKEN=
DEBUG=false
TYPE=prod
LOG_LEVEL=info
```

- Run the server locally (and watch changes):

```bash
$ npm start
```
