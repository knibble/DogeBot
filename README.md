## Development

### Getting Started

#### Install dependencies

```bash
yarn
```

#### Add an .env file with this setup:

```bash
COMMAND_PREFIX=!
OWNER=
BOT_ID=
BOT_TOKEN=
DEBUG=false
TYPE=prod
LOG_LEVEL=info
```

- Run the server locally (and watch changes):

```bash
$ yarn start
```

## Running on Docker

```bash
docker-compose up
```
